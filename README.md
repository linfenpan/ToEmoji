# emoji 整理

时间: 2015/08/25

暂时整理的了 842 个 emoji 表情，不过最后图标的，好像有点乱掉了。暂时不理。

这里，提供了两种修复 PC 端 emoji 显示的方法：

 1. font-face 引入外部字体库
 2. ToEmoji.js 通过 emoji 配置文件，对html内容，进行图片替换文字的操作。


各有各的有点，font-face某些程度上，可能效率更高。

ToEmoji.js在小型场景下，可节省更多流量。
